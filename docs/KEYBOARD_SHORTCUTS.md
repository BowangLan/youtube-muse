# Keyboard Shortcuts

YouTube Muse supports comprehensive keyboard shortcuts for efficient playback control without touching your mouse.

## Playback

| Key | Action |
|-----|--------|
| `Space` or `K` | Play/Pause |
| `N` | Next track |
| `P` | Previous track |
| `→` or `L` | Skip forward 10s |
| `←` or `J` | Skip backward 10s |

## Volume

| Key | Action |
|-----|--------|
| `↑` | Increase volume 5% |
| `↓` | Decrease volume 5% |
| `M` | Mute/Unmute |

## Navigation

| Key | Action |
|-----|--------|
| `1-9` | Play intent card 1-9 |

## Help

| Key | Action |
|-----|--------|
| `?` | Show keyboard shortcuts |

## Notes

- Shortcuts are disabled when typing in input fields
- YouTube-style shortcuts (`K`, `J`, `L`) mirror familiar controls
- Intent shortcuts map to visible cards in order (built-in + custom)
